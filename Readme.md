# Bear Bot AI Assistant

一个基于 AI 的聊天助手，支持文件解析和对话。

## 功能特点

- 用户注册和登录
- AI 对话
- 文件解析
- 聊天历史记录
- 响应式设计

## 技术栈

### 前端
- React
- Vite
- React Router
- CSS3

### 后端
- Node.js
- Express
- MySQL
- Redis
- JWT

## 安装和运行

### 前端
```bash
cd frontend
npm install
npm run dev
```

### 后端
```bash
cd backend
npm install
npm run dev
```

## 环境变量配置

请参考 `.env.example` 文件配置必要的环境变量。
